## Usage
Install what you need:
```
$ npm install
```
And then you can run it:
```bash
$ npm test # single-run debug task, requires devDependencies
$ npm debug # full all in debug mode
$ npm start # full app
```

> **Note:** nodeBT runs (now) in port 8081
